# Security
If you believe you've found a security vulnerability, please post an issue
on the [GitLab](https://gitlab.com/groups/Elypia/-/issues) repository and
mark it as a [confidential issue](https://docs.gitlab.com/ce/user/project/issues/confidential_issues.html).

[GitLab](https://gitlab.com/) is itself part of the open-source community
and can be self-hosted should the need ever arrise.

## Questions
If you have questions around security but are unsure or haven't
pin pointed anything specific, you're welcome to contact us.

* Send an email to security@elypia.org.
